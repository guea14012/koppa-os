Getting Started

Boot Guide

Three ways to run KOPPA-OS — USB live boot, VirtualBox, or VMware.

USB BOOT

Flash to USB Drive

Requires a USB drive 4GB+. Use Balena Etcher (Windows/Mac/Linux) or dd on Linux.

dd if=koppa-os-*.iso of=/dev/sdX bs=4M status=progress

Boot from USB → select "Live" → login: root / koppa

VIRTUALBOX

Run in VirtualBox

New VM → Linux/Debian 64-bit → 4GB RAM → 2 CPU → attach ISO as optical disk → Boot.

Recommended: 4096 MB RAM · 2 vCPU · NAT + Host-only adapter
VMWARE

Run in VMware

New VM → Linux → Debian 12 → attach ISO → Power on. Enable 3D acceleration for best performance.

Recommended: 4096 MB RAM · 2 vCPU · Bridged network
VERIFY

Verify ISO Integrity

Always verify the SHA256 checksum before booting to ensure the ISO was not tampered with.

sha256sum koppa-os-*.iso

Compare with the checksum shown in the Download section below.
